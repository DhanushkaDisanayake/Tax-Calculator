# Tax Calculator System (Java)

This is a Java console-based Tax Calculator System developed as a Programming Fundamentals coursework.  
The program allows users to calculate various taxes (Withholding, Payable, Income, SSCL) and leasing payments using loops, conditions, and switch cases.

### 4. SSCL (Social Security Contribution Levy)

### 5. Leasing Payment

- Monthly installment calculation
- Leasing category search (36 / 48 / 60 months)
